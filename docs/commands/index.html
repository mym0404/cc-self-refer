Commands

Claude Code commands that integrate with the cc-self-refer CLI tool.

Available Commands​

Page Management​

  • /page-refer - Load session history context
  • /page-save - Save current session history

Pattern Management​

  • /pattern-create - Create reusable code pattern
  • /pattern-use - Apply existing code patterns

Plan Management​

  • /plan-create - Create strategic development plan
  • /plan-edit - Edit existing plan
  • /plan-resolve - Resolve and complete plans

Specification Management​

  • /spec - Create comprehensive project specification
  • /spec-refer - Reference project specifications

Getting Started​

Each command integrates with the cc-self-refer CLI tool. Make sure you have initialized your project:

cc-self-refer init

For detailed usage of individual commands, click on the command links above.
